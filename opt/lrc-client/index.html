How to add server

For add a server (computer that will be controled), you need know the IP address of this computer. For discover the IP address of your computer, type the following command on your terminal.


ifconfig


Now that you have the IP address of your computer, just go to "Add server" page, on home screen and type the name of server (any name of your choice) and the IP address of your computer and tap in "Save" button. This is it.

How to install on GNU/Linux

If you are with app in hands, it is likely that you have already installed it on your GNU/Linux. But if somehow you yet no have already installed, follow the following steps.


1. Download the lrc on you GNU/Linux

wget http://www.linuxremotecontrol.com/lrc.deb


2. Install .deb package

sudo dpkg -i lrc.deb


3. Move /opt/lrc-client directory to your device.

sudo mv /opt/lrc-client your-device-directory


4. Start lrc-server

node /opt/lrc-server/lrc.js


5. Open the index.html of your-device-directory/lrc-client in a browser, add your server and have fun
